Livepal
macOS 14+ On-device speech

Live captions for bilingual calls.

Capture incoming meeting audio from a window you choose, transcribe on your Mac, and show two caption lanes for languages you pick — optimized when speakers take turns.

Download for macOS View source

One click downloads Livepal-macos.zip. Unzip, drag Livepal.app to Applications, then open it. First launch: allow Screen Recording and Speech Recognition in System Settings.

If the file is not published yet, open Releases or build locally: cd macos && ./scripts/package_app.sh.

Incoming audio

Ten languages

Configure two lanes from ten supported locales; routing uses on-device language hints.

No API bill

Designed for free end-user use: on-device Speech when supported for your locales.

Turn-taking
